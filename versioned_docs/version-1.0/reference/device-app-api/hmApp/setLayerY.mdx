Scroll to a particular set of coordinates in the current page

## Type

```ts
(offset: number) => void
```

## Parameters

### offset

|      Description    | Required |    Type  | Default |
| ------------------- | -------- | -------- | ------ |
| y-coordinate offset |   YES    | `number` |   -    |

## Code example

```js
// Set the y-coordinate offset
hmApp.setLayerY(100)
```
