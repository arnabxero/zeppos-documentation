sidebar_label: setStatusBarVisible

This interface is only available on square screen devices, set the status bar visible or not.

For square screen title bar, refer to [Screen Adaptation](../../../guides/best-practice/multi-screen-adaption.mdx).

## Type

```ts
(visible: boolean) => void
```

## Parameters

| parameter | description | type |
| ------- | --------------------------------------- | --------- |
| visible | `true`: show the status bar; `false`: hide the status bar | `boolean` |

## Code examples

```js
hmUI.setStatusBarVisible(false)
```
