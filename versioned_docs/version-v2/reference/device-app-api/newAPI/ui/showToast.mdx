![show_toast](/img/api/show_toast.jpg)

Show Toast with `\n` text line feed support.

## Type

```ts
(option: Option) => void
```

## Parameters

### Option: object

| Properties | Description                |   Type   |
| ---------- | -------------------------- | -------- |
|    text    | The text content of toast. | `string` |

## Code examples

```js
import { showToast } from '@zos/ui'

Page({
  build() {
    showToast({
      text: 'Hello\nZepp OS'
    })
  }
})
```
