# 关键词

基于「美好」「健康」和「有温度」的理念，我们描绘出生活场景：透过半透明玻璃的柔和的光线，投射在玻璃上模糊的色彩和斑驳的倒影。

![Design](/img/design/light_shine_through_glasses.png)

从这些场景里，我们提取出3个视觉关键词：磨砂玻璃 • 光影 • 色彩

![Design](/img/design/frostedglass_lightandshadow_colors.png)

将包含「温暖」特质的「光」进行可视化表达：当半透明的玻璃质感对象置于顶层时，所产生的色彩变化、光影折射、空间表达，让 Zepp OS 的视觉更贴近生活和自然。
