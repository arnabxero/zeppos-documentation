## 最新版本

:::info
当前 Zepp OS 最新版本是 3.0
:::

## 文档版本说明

Zepp OS 文档版本与 Zepp OS 版本保持一致，v3 对应 Zepp OS 3.x 版本的内容，点击文档平台左上角版本切换按钮可以查看早期 Zepp OS 版本的文档。

![version_change](/img/version_change.gif)

:::tip
如果开发者想为搭载 Zepp OS 1.0 的设备开发应用，请查看 v1.0 版本的文档。

想为搭载 Zepp OS 2.x 及以上的版本的设备开发应用，直接查看最新版本的文档。

Zepp OS 2.0 引入了 API_LEVEL 的概念，后续的系统版本会保证向下兼容，兼容性相关参考 [《API_LEVEL（兼容性）》](../guides/framework/device/compatibility.md)。
:::

:::info
查看搭载 Zepp OS 的设备的最新系统版本请参考 [搭载 Zepp OS 设备列表](../reference/related-resources/device-list.mdx)。
:::
