在这一步中，我们使用 Zeus CLI 命令行工具完成项目创建

## 使用 Zeus CLI 创建模版项目

```sh
zeus create hello-world
```

<img src={useBaseUrl('/img/docs/quick-start/cli_interaction.gif')} width="100%" title="小程序应用商店" />

根据上图指示，进行命令行交互选择。

命令执行成功后，可以看到 `/hello-world` 目录，小程序项目创建成功。
