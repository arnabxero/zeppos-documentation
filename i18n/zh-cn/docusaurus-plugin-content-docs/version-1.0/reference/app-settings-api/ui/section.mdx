## 类型

```ts
(props: Props, renderFuncArr?: RenderFunc | Array<RenderFunc>) => result: RenderFunc
```

## Props: object

| 名称        | 说明                    | 必填 | 类型     | 默认值 |
| ----------- | ----------------------- | ---- | -------- | ------ |
| style       | 样式属性，支持 CSS 属性 | 否   | `object` | -      |
| title       | Section 的标题          | 否   | `string` | -      |
| description | Section 的描述          | 否   | `string` | -      |
