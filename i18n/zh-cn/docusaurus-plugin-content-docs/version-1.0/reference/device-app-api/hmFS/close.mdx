> Zepp OS 小程序文件系统介绍请参考 [文件系统](../../../guides/framework/device/fs.md)。

关闭文件句柄。

## 类型

```ts
(fileId: number) => result
```

## 参数

### fileId

| 说明                     | 类型     |
| ------------------------ | -------- |
| 文件句柄，打开文件时返回 | `number` |

### result

| 说明               | 类型     |
| ------------------ | -------- |
| 结果，`0` 获取成功 | `number` |

其余错误码请参考 [文件系统 - 错误代码](../../../guides/framework/device/fs.md#错误代码)。

## 代码示例

```js
Page({
  build() {
    const fileId = hmFS.open('test_file.txt', hmFS.O_RDWR | hmFS.O_CREAT)

    //关闭文件句柄
    hmFS.close(fileId)
  }
})
```
