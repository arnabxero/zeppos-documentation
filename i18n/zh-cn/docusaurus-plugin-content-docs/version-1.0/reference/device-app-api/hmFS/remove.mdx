> Zepp OS 小程序文件系统介绍请参考 [文件系统](../../../guides/framework/device/fs.md)。

删除小程序 `/data` 数据目录下文件。

## 类型

```ts
(path: string) => result
```

## 参数

### path

| 说明                                    | 必填 | 类型     | 默认值 |
| --------------------------------------- | ---- | -------- | ------ |
| 小程序的 `/data` 目录下文件名(相对路径) | 是   | `string` | -      |

### result

| 说明                   | 类型     |
| ---------------------- | -------- |
| 操作结果，`0` 表示成功 | `number` |

其余错误码请参考 [文件系统 - 错误代码](../../../guides/framework/device/fs.md#错误代码)。

## 代码示例

```js
Page({
  build() {
    const result = hmFS.remove('path/to/test_file.txt')
    console.log(result)
  }
})
```
