设置当前页面滚动条的纵坐标位置

## 类型

```ts
(offset: number) => void
```

## 参数

### offset

| 说明       | 必填 | 类型     | 默认值 |
| ---------- | ---- | -------- | ------ |
| 页面的纵轴坐标 | 是   | `number` | -      |

## 代码示例

```js
// 页面滚动到纵坐标为 100px 的位置
hmApp.setLayerY(100)
```
