当前 screen 重新加载页面

## 类型

```ts
(option: Option) => void
```

## 参数

### Option

| 参数  | 说明     | 必填 | 类型     | 默认值 |
| ----- | -------- | ---- | -------- | ------ |
| url   | 跳转路径 | 是   | `string` | -      |
| param | 参数     | 否   | `string` | -      |

## 代码示例

```js
// 加载pages/index3.js页面
hmApp.reloadPage({ url: 'pages/index3', param: '...' })
```
