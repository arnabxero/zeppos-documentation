系统：Windows 10 


## 时间一览表

|                           任务项                          |            用时       |
|  -------------------------------------------------------- | -------------------- |
| Vmware安装                                                |   2min               |
| 虚拟机安装                                                 |   50min             |
| 虚拟机环境配置（包括安装模拟器）                             |   26min             |
| 环境安装(node.js 和 npm i @zeppos/zeus-cli -g)             |   10min             |
| 本机VScode安装调试                                         |   5min              |
| 创建工程                                                   |  1min               |
| 虚拟机上模拟器配置                                          |   2min              |
| 模拟器预览                                                 |   1min               |
| 手机预览                                                  |   1min                |
| **总用时**                                                |  98min(1小时38分钟)   |



## 备注

- 表格中的每一步都是按照指导文档上的步骤进行操作，相对来说比较节省时间。
- 软件的下载时间均不计算在内。
- 计时器是手机秒表，不足1min按1min计算。
- 使用`npm`安装`zeus-cli`时，有时会因网络问题导致安装出错，需要重复安装几次，会耗时久一点。倘若问题一直没解决，可以先执行`npm install -g cnpm --registry=https://registry.npm.taobao.org`，再执行`cnmp install`就可以了。
- 以上环境配置及安装可能会受到网络环境等因素影响，实际效果因人而异，仅供参考。
