# 进度

进度指示器通过向用户反馈当前的响应进度和合理的时间消耗，指引用户顺利的完成目标任务，使用户的等待过程变得清晰而流畅。

## 设计目标

- 帮助用户明确当前状态，为用户的后续决策提供充足的支持；
- 向用户传达有根据的等待原因，缓解用户等待时的焦虑感；
