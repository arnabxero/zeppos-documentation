Wearable devices with Zepp OS usually have physical keys, you can find the key layout and key names in this document

---|--|--

## Four Key

![4 key](/img/docs/guides/related-resources/4key.png)

Location|Name|Digital Crown
---|--|--

2|DOWN|NO
3|SELECT|NO
4|BACK|NO
