# resetPageBrightTime

> Start from API_LEVEL `2.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Cancel the bright time set by `setPageBrightTime`.

## Type

```ts
function resetPageBrightTime(): Result
```

## Parameters

### Result

| Type                | Description                              |
| ------------------- | ---------------------------------------- |
| <code>number</code> | If `0` is returned, success is indicated |

## Example

```js
import { setPageBrightTime, resetPageBrightTime } from '@zos/display'

setPageBrightTime({
  brightTime: 60000,
})

const result = resetPageBrightTime()
```
