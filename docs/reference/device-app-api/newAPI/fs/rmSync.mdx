# rmSync

> Start from API_LEVEL `2.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Synchronously delete files in the `/data` directory of the Mini Program.

## Type

```ts
function rmSync(option: Option): Result
```

### Simplified calling method

```ts
function rmSync(path: string): Result
```

## Parameters

### Option

| Property | Type                | Required | DefaultValue | Description | API_LEVEL |
| -------- | ------------------- | -------- | ------------ | ----------- | --------- |
| path     | <code>string</code> | Y        | -            | path        | 2.0       |

### Result

| Type                | Description                              |
| ------------------- | ---------------------------------------- |
| <code>number</code> | If `0` is returned, success is indicated |

## Example

```js
import { rmSync } from '@zos/fs'

const result = rmSync({
  path: 'test.txt',
})

if (result === 0) {
  console.log('rmSync success')
}
```
