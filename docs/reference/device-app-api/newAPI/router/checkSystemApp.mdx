# checkSystemApp

> Start from API_LEVEL `3.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Check if the system application supports jumping.

## Type

```ts
function checkSystemApp(option: Option): void
```

### Simplified calling method

```ts
function checkSystemApp(appId: number): void
```

## Parameters

### Option

| Property | Type                | Required | DefaultValue | Description                                                                      | API_LEVEL |
| -------- | ------------------- | -------- | ------------ | -------------------------------------------------------------------------------- | --------- |
| appId    | <code>number</code> | Y        | -            | ID of the system App to be jumped to, value refers to the system App ID constant | 3.0       |
