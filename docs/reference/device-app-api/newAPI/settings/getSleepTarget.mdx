# getSleepTarget

> Start from API_LEVEL `2.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Get the sleep target set by the user.

## Type

```ts
function getSleepTarget(): Result
```

## Parameters

### Result

| Type                | Description                                       |
| ------------------- | ------------------------------------------------- |
| <code>number</code> | User-set sleep target, default is `0`, in minutes |

## Example

```js
import { getSleepTarget } from '@zos/settings'

const sleepTarget = getSleepTarget()
console.log(sleepTarget)
```
