# start

> Start from API_LEVEL `3.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Start the specified App service, return the result through the callback function.

:::info
permission code: `device:os.bg_service`
:::

## Type

```ts
function start(option: Option): Result
```

## Parameters

### Option

| Property      | Type                                                      | Required | DefaultValue | Description                                                                              | API_LEVEL |
| ------------- | --------------------------------------------------------- | -------- | ------------ | ---------------------------------------------------------------------------------------- | --------- |
| file          | <code>string</code>                                       | Y        | -            | The App Service js file must be the one configured in the module app-service in app.json | 3.0       |
| param         | <code>string</code>                                       | N        | -            | Parameters passed in when the js file is loaded by the backend service                   | 3.0       |
| complete_func | <code>(callbackOption: CallbackOption) =&#62; void</code> | Y        | -            | Callback function for the completion of the backend service start                        | 3.0       |

### CallbackOption

| Property | Type                 | Description                                                           | API_LEVEL |
| -------- | -------------------- | --------------------------------------------------------------------- | --------- |
| file     | <code>string</code>  | App service js file, same as `start` incoming parameters              | 3.0       |
| result   | <code>boolean</code> | App service start result, `true` means success, `false` means failure | 3.0       |

### Result

| Type                 | Description                                                   |
| -------------------- | ------------------------------------------------------------- |
| <code>boolean</code> | If `0` is returned, The App Service was started successfully. |

### ERROR_CODE

| Value | Type                | Description                                         | API_LEVEL |
| ----- | ------------------- | --------------------------------------------------- | --------- |
| 0     | <code>number</code> | Success                                             | 3.0       |
| 1     | <code>number</code> | Parameter error                                     | 3.0       |
| 2     | <code>number</code> | Service Status Error                                | 3.0       |
| 3     | <code>number</code> | No Permission                                       | 3.0       |
| 4     | <code>number</code> | Out Of Memory                                       | 3.0       |
| 5     | <code>number</code> | Not Supported                                       | 3.0       |
| 6     | <code>number</code> | Prohibited                                          | 3.0       |
| 7     | <code>number</code> | The number of services has reached the system limit | 3.0       |
| 255   | <code>number</code> | Unknown Error                                       | 3.0       |

## Example

```js
import { start } from '@zos/app-service'
```
