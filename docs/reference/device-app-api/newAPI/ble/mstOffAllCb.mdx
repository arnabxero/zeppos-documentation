# mstOffAllCb

> Start from API_LEVEL `3.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Unregister of all registered Bluetooth-related callback functions.

## Type

```ts
function mstOffAllCb(): void
```

## Example

```js
import { mstOffAllCb } from '@zos/ble'

mstOffAllCb()
```
