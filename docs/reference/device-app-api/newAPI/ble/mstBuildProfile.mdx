# mstBuildProfile

> Start from API_LEVEL `3.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Creating a Profile connection.

## Type

```ts
function mstBuildProfile(profile: ProfileObj): Result
```

## Parameters

### ProfileObj

| Property | Type                                    | Required | DefaultValue | Description                                                   | API_LEVEL |
| -------- | --------------------------------------- | -------- | ------------ | ------------------------------------------------------------- | --------- |
| pair     | <code>boolean</code>                    | Y        | -            | Whether to pair automatically                                 | 3.0       |
| id       | <code>number</code>                     | Y        | -            | Connection ID                                                 | 3.0       |
| profile  | <code>string</code>                     | Y        | -            | Profile Name                                                  | 3.0       |
| dev      | <code>ArrayBuffer</code>                | Y        | -            | Device MAC address, 6 bytes long, Uint8Array view recommended | 3.0       |
| len      | <code>number</code>                     | Y        | -            | `list` array length                                           | 3.0       |
| list     | <code>Array&#60;ServicesObj&#62;</code> | Y        | -            | Services list array                                           | 3.0       |

### ServicesObj

| Property | Type                                   | Required | DefaultValue | Description         | API_LEVEL |

| list     | <code>Array&#60;ServiceObj&#62;</code> | Y        | -            | Service array       | 3.0       |
