# mstOnCharaValueArrived

> Start from API_LEVEL `3.0` . Please refer to [API_LEVEL](../../../../guides/framework/device/compatibility.md).

Register to read Characteristic data to the callback function.

## Type

```ts
function mstOnCharaValueArrived(callback: Callback): Result
```

## Parameters

### Callback

| Type                                                                                | Description                                       |
| ----------------------------------------------------------------------------------- | ------------------------------------------------- |
| <code>(profile: Profile, uuid: UUID, data: Data, status: Status) =&#62; void</code> | Read Characteristic data to the callback function |

### Profile

| Type                | Description     |
| ------------------- | --------------- |
| <code>number</code> | Profile pointer |

### UUID

| Type                | Description                |
| ------------------- | -------------------------- |
| <code>string</code> | Characteristic UUID string |

### Data

| Type                     | Description                              |
| ------------------------ | ---------------------------------------- |
| <code>ArrayBuffer</code> | Reads the data using the Uint8Array view |

### Status

| Type                | Description                   |
| ------------------- | ----------------------------- |
| <code>number</code> | Status, `0` indicates success |

### Result

| Type                 | Description                                                                  |
| -------------------- | ---------------------------------------------------------------------------- |
| <code>boolean</code> | The result of the function call, `true` means success, `false` means failure |
